# Pinecone RAG Chatbot (LangChain + FastAPI + Streamlit)

A beginner-friendly Retrieval-Augmented Generation (RAG) chatbot project using:

- **LangChain** for RAG pipeline
- **Hugging Face embeddings** (sentence-transformers)
- **Pinecone** as the vector database
- **FastAPI** for backend API
- **Streamlit** for frontend chat UI

> Deployment: Streamlit Cloud runs the **frontend**. Deploy the **backend** separately (Render/Railway/Fly.io), then connect Streamlit to it via `BACKEND_URL`.
